## 📋 Description du projet

Ce projet est un **CV en ligne interactif et responsive** réalisé en HTML, CSS et JavaScript. Il présente mon parcours académique, mes expériences professionnelles, mes compétences techniques et personnelles, ainsi qu'un formulaire de contact fonctionnel.

Le design adopte un thème **dark emerald / or** avec une typographie éditoriale (Cormorant Garamond + Outfit) pour un rendu professionnel et distinctif.

---

## 🛠️ Technologies utilisées

| Technologie | Usage |
|-------------|-------|
| **HTML5** | Structure sémantique du contenu |
| **CSS3** | Mise en forme, variables CSS, animations |
| **JavaScript (ES6)** | Formulaire de contact, animations au scroll |
| **Bootstrap 5.3** | Grid responsive, composants UI |
| **Bootstrap Icons** | Icônes vectorielles |
| **Google Fonts** | Cormorant Garamond + Outfit |

### Structure des fichiers

```
cv-online-mourad/
├── index.html        # Structure principale du CV
├── style.css         # Thème, variables CSS, animations
├── script.js         # Logique JS (formulaire + scroll)
└── README.md         # Documentation du projet
```
